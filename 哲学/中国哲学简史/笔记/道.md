# 二 老子

- **道**：
	- [[名|名家]]**启发**：
		- 至大无外
		- 共相的独立性
	- “==**无名之名**==”：
		**“道”是无从命名的**，而万物之名又都是由道而来
- **常**：
	万物都是变动不居的，但决定万物变动的法则却是不变的
- **德**：
	- <mark style="background: #FF5582A6;">万物背后都有一个无法解释的规则，指导万物运行</mark>
		==在万物之中的“道”就是“德”== 
		
	- <mark style="background: #FF5582A6;">道家则是去学习道德，利用道德改善自身，全性保真</mark> 
- **方法论**：
	- “无为”：
		“不刻意为之”

# 三 庄子

- **有限的观点**：
	- 认识到<mark style="background: #FF5582A6;">人的认识是有限的</mark>：
		庄子则认为，**人们的是非观念是根据他们的局限性观点建立起来的**。所有这样的观点都是相对的
- **相对快乐**：（==在自我局限内“**各尽所能**”==）
	- 让万物自由发挥天赋才能，即 “德”
- **绝对快乐**：（==在**更高层次**看待事物==）
	- 超越个体局限、与宇宙合一的绝对快乐
	- 至此，<mark style="background: #FF5582A6;">从哲学上，将全性保身解决</mark>：
		==如果人达到与万物一体，这时，人的肢体无非是尘埃；生死终始，无非是日夜的继续，不足以干扰人内心的宁静；至于世俗的得失、时运好坏，更不足挂齿==

# 四 理性的新道家

> 魏晋时期（公元 3-4 世纪）兴起的 “玄学”（又称 “新道家”），以**向秀、郭象**的 《**庄子注**》为核心

- 以==更具体的语言，解释《庄子》中的思想==：
	“曾见郭象注庄子，识者云：却是庄子注郭象”
- **独化**：
	- ==造物者无主，而物各自造==：
		**“道” 不是造物主，而是物自生自造** 
		万物的生成是自身本性的自然展开，**无需依赖任何外在条件** 
- **方法论**：
	- 消除 ”**偏尚之累**“：
		- 消除时刻苦于选此舍彼的烦恼，认识到万物本质上并无差别
	- **弃彼任我**：
		人与人、事与事是不同的，变化的，模仿则抛弃了自我，抛弃天性
	- <mark style="background: #FF5582A6;">融入万物的“独化”</mark>：
		**不仅仅**跳出万物，在更高层次看待万物，而是要==进一步发挥**融入自身乃至万物的 ”独化“**==
