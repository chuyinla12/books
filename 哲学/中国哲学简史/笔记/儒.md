# 五 至汉以来的发展

## （一）形而上学发展

- **道**：
	- 儒和道区别：
		- ==儒家==的道是**指某一事物的规律**（更接近现在人们理解的道）
		- ==道家==的道是**万物之前的 “无”** 
	- [[中国哲学简史#万物变易之“道”|万物之道]]：
		- 万物==既有自身之道，整体也有道== 
		- **符号**：
			- 易有太极，是生两仪，两仪生四象，四象生八卦
			- 一阴一阳之谓道
			这里 **”易“ 和 ”道“** 可以互换
			**“两仪” 和 “阴阳”** 可以互换
- **《中庸》**：
	- 中：恰如其分
	- 和：差异的协调
	- 庸：道的一般性，普遍性
	- 明：觉悟
		- <mark style="background: #FF5582A6;">一个人如果力求完善自己，他就会看到，为此也必须同时完善他人</mark> 
		- <mark style="background: #FF5582A6;">明即达到 “天人合一” 的境界</mark> 
	- 诚：纯粹无伪

## （三）理学、心学

### 1 起源

周敦颐的两个学生，程颢、程颐
- **程颐**：
	- 由朱熹集大成
	- 史称 ==“程朱学派” 或 “理学”== 
- **程颢**：
	- 由陆象山和王守仁完成
	- 史称 ==“陆王学派” 或 “心学”== 

### 2 对比

|          | 程朱理学                                              | 陆王心学                         |
| -------- | ------------------------------------------------- | ---------------------------- |
| **理**    | 客观、独立的精神实体，是宇宙与伦理的 “本源”<br>                       | 主观、内在的 “本心”，是人心固有的 “良知”      |
| **气**    | 重要但非本源，是 “理” 的 “载体”                               | 次要的，是 “心” 的外在表现              |
| **关系**   | **理先气后**<br>**理气二分**：<br> 理是精神（形而上）<br> 气是物质（形而下） | **心物合一**：<br>“物” 的意义由 “心” 赋予 |
| **核心命题** | ==理在气先，格物致知==                                     | ==心即理，致良知==                  |
| **求理路径** | ==向外==：研究事物→积累知识→明理                               | ==向内==：反省本心→去除私欲→显良知         |
